Web Assembly

Web Assembly or WASM is the binary instruction generated from high level languages such as Go, C, C++ or Rust. It is faster than JavaScript and WASM 1.0 has already shipped in the major browsers. It is being touted by someone to eventually replace JavaScript but I seriously doubt that it would happen or see it happening anytime soon
